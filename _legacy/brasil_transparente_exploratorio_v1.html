Glossário cidadão

Explicações curtas para reduzir ruído e aumentar compreensão.

Dívida Bruta

Estoque total de obrigações do governo geral. É um dos principais indicadores fiscais acompanhados pelo Banco Central.

Juros da dívida

Custo financeiro pago para carregar a dívida pública. Não é a mesma coisa que amortização.

LOA

Lei Orçamentária Anual. Define quanto o governo está autorizado a gastar e em quais áreas.

ITD

Índice de Transparência dos Dados: nota interna de 0 a 100 baseada em fonte, atualização, metodologia, histórico e download.

Por que confiar?

O portal precisa ser transparente até sobre suas limitações. Este bloco mostra o padrão mínimo para cada número publicado.

Governança dos dados

1. Fonte visível

Todo KPI deve exibir órgão, competência, frequência e link oficial.

2. Metodologia aberta

O usuário precisa entender como o cálculo foi feito, inclusive quando for estimativa.

3. Dados exportáveis

CSV, JSON e API reduzem dependência da interface e aumentam auditabilidade.
